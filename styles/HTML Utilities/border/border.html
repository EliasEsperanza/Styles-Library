Borders

Utilice utilidades de bordes para diseñar rápidamente el borde y el radio del borde de un elemento. Ideal para imágenes, botones o cualquier otro elemento.

Border

Utilice utilidades de bordes para agregar o eliminar los bordes de un elemento. Elija entre todos los bordes o uno a la vez.

.brd-top-Electric
.brd-left-Poison
.brd-right-Ground
.brd-top-Dark
