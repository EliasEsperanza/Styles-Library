Button tags

Las clases .btn están diseñadas para usarse con el elemento "button". Sin embargo, también puedes usar estas clases en elementos "a" o "input" (aunque algunos navegadores pueden aplicar una representación ligeramente diferente).

Cuando se utilizan clases de botones en elementos "a" que se utilizan para activar la funcionalidad en la página (como contraer contenido), en lugar de vincular a nuevas páginas o secciones dentro de la página actual, a estos vínculos se les debe asignar un rol="botón" para transmitir su propósito a tecnologías de asistencia como lectores de pantalla.

Normal Ice Psychic Dark 
            <a class="btn btn-ButtonTag-Normal" href="#" role="button">Normal</a>;
            <button class="btn btn-ButtonTag-Fire" type="submit">Fire</button>;
            <input class="btn btn-ButtonTag-Water" type="button" value="Water">;
            <input class="btn btn-ButtonTag-Grass" type="submit" value="Grass">;
            <input class="btn btn-ButtonTag-Electric" type="reset" value="Electric">;

Outline buttons

¿Necesitas un botón, pero no los fuertes colores de fondo que traen? Reemplace las clases modificadoras predeterminadas con las .btn-outline-* para eliminar todas los colores de fondo para agregar un diseño diferente y mas interactivo.
