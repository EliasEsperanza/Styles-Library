Badges

Aprender de como usar y a donde usarlos didferentes tamaños de h* y los diferentes colores que ofercemos de los 18 tipos de pokemon

H*

Puedes utilizar los diferentes tamaños de h* que existe disponibles y te explicamos para que y como usarlo

  • H1 Título principal
    Se utiliza para el título más importante de una página, como el encabezado principal o el título de una sección principal.
  • H2 Título secundario
    Se utiliza para títulos de secciones secundarias, subsecciones importantes o encabezados de bloques de contenido.
  • H3 Título terciario
    Se usa para títulos de subsecciones dentro de secciones principales o para encabezados de bloques de contenido menos importantes.
  • H4 Título cuaternario
    Se utiliza para títulos de subsecciones adicionales, encabezados de bloques de contenido menos relevantes o para etiquetas específicas dentro de un contenido.
  • H5 Título quinario
    Se usa para títulos de subsecciones muy específicas o etiquetas dentro de un contenido detallado.
  • H6 Título sextario
    Se utiliza para títulos de subsecciones extremadamente específicas, etiquetas muy específicas dentro de un contenido detallado o elementos de navegación muy específicos.
