Containers

Los contenedores en términos de desarrollo web se refieren a elementos que se utilizan para agrupar y organizar contenido dentro de una página.

Containers

PokeStrap viene con 5 tipos de contenedores diferentes

  • breakpoint-small: 576px;
  • breakpoint-medium: 768px;
  • breakpoint-large: 992px;
  • breakpoint-extra-large: 1100px;
  • breakpoint-extra-extra-large: 1400px;

Breakpoint-small: 576px

Este tamaño de contenedor es ideal para dispositivos móviles pequeños, como teléfonos inteligentes en posición vertical. Es el punto en el que comienza a aplicarse el diseño responsivo para dispositivos móviles.
