Range

Utilice nuestras entradas de rango personalizado para lograr un estilo uniforme en todos los navegadores y una personalización integrada.

Ejemplos

Descripción general Cree controles <input type="range"> personalizados con .form-range-* inspirados de cada uno de los tipos de Pokemon.

< label for="customRange1" class="form-label">
Example range Normal
</label>
<input type= "range" class= "form-range-Normal" id= "customRange1">   

< label for="customRange1" class="form-label">
Example range Normal
</label>
<input type= "range" class= "form-range-Fire" id= "customRange1">

Disabled

Agregue el atributo disabled (deshabilitado) en el input del range para bloquear el uso del range.

< label for="customRange1" class="form-label">
Example range Normal
</label>
<input type= "range" class= "form-range-Normal" id="customRange1"
disabled>

Min and max

Las entradas de rango tienen valores implícitos para mínimo y máximo: 0 y 100, respectivamente. Puede especificar nuevos valores para aquellos que usan los atributos mínimo y máximo.

< label for="customRange1" class="form-label">
    Example range Normal
</label>
<input type="range" class="form-range-Normal" id="customRange1" 
min="0" max="5" disabled>

Steps

De forma predeterminada, las entradas de rango se “ajustan” a valores enteros. Para cambiar esto, puede especificar un valor de salto. En el siguiente ejemplo, duplicamos el número de pasos usando step="0.5".

< label for="customRange1" class="form-label">
    Example range Normal
</label>
<input type="range" class="form-range-Normal" id="customRange1" 
min="0" max="5" step="0.5" disabled>
