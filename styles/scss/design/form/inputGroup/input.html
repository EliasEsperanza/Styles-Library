Input group

Amplíe fácilmente los controles de formulario agregando inputs, texto y botones a ambos lados de las entradas de texto, selecciones personalizadas y entradas de archivos personalizados.

Basic example

Coloque un complemento o botón a cada lado de una entrada. También puedes colocar uno a ambos lados de una entrada. Recuerde colocar los <label>s fuera del grupo de entrada.

Checkboxes and radios

Coloque cualquier casilla de verificación u opción de radio dentro del complemento de un grupo de entrada en lugar de texto. Recomendamos agregar .form-check-input-*-Default o .form-radio-input-*-Default cuando no haya texto visible al lado de la entrada.

<div class="input-group-Normal ">
  <div class="input-group-text">
    <div class="form-check-Normal-Default">
      <input class="form-check-input" type="checkbox" value="" 
      id="customCheckbox">
    </div>
  </div>
  <input type="text" aria-label="Text input with checkbox">
</div>

<div class="input-group-Normal">
  <div class="input-group-text">
    <div class="form-Radio-Normal-Default">
      <input class="form-check-input" type="radio" 
      name="flexRadioDefault" id="customRadio">
    </div>
  </div>
  <input type="text" aria-label="Text input with radio button">
</div>

Multiple inputs

Si bien se admiten visualmente varios <input>, los estilos de validación solo están disponibles para grupos de entrada con un único <input>. Si quieres agregar mas input agrega la clase "form-grupo" para que se añada un diseño diferente

First and last name 
  <div class="input-group-Normal">
    <span class="input-group-text">First and last name</span>
    <input type="text" aria-label="First name">
    <input type="text" aria-label="Last name" class="form-grupo">
</div>

Buttons addons

Se admiten añadir botones al inicio y final del input.

<div class="input-group-Normal">
  <button class="btn btn-Normal btn-Inicio" type="button" 
  id="button-addon1">Button</button>
  <input type="text" placeholder="" 
  aria-label="Example text with button addon" 
  aria-describedby="button-addon1">
</div>

<div class="input-group-Normal">
  <input type="text" placeholder="Recipient's username" 
  aria-label="Recipient's username" aria-describedby="button-addon2">
  <button class="btn btn-Normal btn-Final" type="button" 
  id="button-addon2">Button</button>
</div>

<div class="input-group-Normal">
  <button class="btn btn-Normal btn-Inicio" 
  type="button">Button</button>
  <button class="btn btn-Normal btn-Inicio" 
  type="button">Button</button>
  <input type="text" placeholder="" 
  aria-label="Example text with two button addons">
</div>

<div class="input-group-Normal">
<input type="text" placeholder="Recipient's username" 
  aria-label="Recipient's username with two button addons">
<button class="btn btn-Normal btn-Final" type="button">Button</button>
<button class="btn btn-Normal btn-Final" type="button">Button</button>
</div>
