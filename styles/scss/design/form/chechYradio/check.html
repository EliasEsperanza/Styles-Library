Checks and Radios

Cree casillas de verificación y radios coherentes en todos los navegadores y dispositivos con nuestros componentes Check y Radio.

Checks

Las casillas de verificación y radios predeterminadas del navegador se reemplazan con la ayuda de .form-check-*-Default o .form-check-*-Checked, una serie de clases para ambos tipos de entrada que mejora el diseño y el comportamiento de sus elementos HTML, que brindan una mayor personalización y coherencia entre navegadores. Las casillas de verificación sirven para seleccionar una o varias opciones en una lista, mientras que las radios sirven para seleccionar una opción entre muchas.

Nuestras comprobaciones utilizan iconos personalizados para indicar estados marcados o indeterminados.

<!--Radio Tipo Normal-->
<div class="form-check-Normal-Default">
    <input class="form-check-input" type="checkbox" 
        value="" id="flexCheckDefault">
    <label class="form-check-label" for="flexCheckDefault">
        Normal Default checkbox
    </label>
</div>

<div class="form-check-Normal-Checked">
    <input class="form-check-input" type="checkbox" 
        value="" id="flexCheckChecked" checked>
    <label class="form-check-label" for="flexCheckChecked">
        Normal Checked checkbox
    </label>
</div>

Check Disabled

Agregue el atributo disabled (deshabilitado) y los check-default no se podran marcar, y los check-checked se mantendran marcados

<!--Radio Tipo Normal-->
<div class="form-check-Normal-Default">
    <input class="form-check-input" type="checkbox" 
        value="" id="flexCheckDefault" disabled>
        <label class="form-check-label" for="flexCheckDefault">
            Normal Default checkbox
        </label>
</div>
            
<div class="form-check-Normal-Checked">
    <input class="form-check-input" type="checkbox" 
        value="" id="flexCheckChecked" checked disabled>
        <label class="form-check-label" for="flexCheckChecked">
            Normal Checked checkbox
        </label>
</div>

Radios

Usa los diferentes tipos de radio para una lista de selecciones, puedes elegir entre form-Radio-*-Default o form-Radio-Normal-Checked

<!--Radio Tipo Normal-->
    <div class="form-Radio-Normal-Default">
        <input class="form-check-input" type="radio" 
         name="flexRadioDefault" id="RadioDefault">
        <label class="form-check-label" for="RadioDefault">
            Normal Default radio
        </label>
    </div>
    
<div class="form-Radio-Normal-Checked">
    <input class="form-check-input" type="radio" 
    name="flexRadioDefaultN" id="RadioChecked" checked>
    <label class="form-check-label" for="RadioChecked">
        Normal Default checked radio
    </label>
</div>

Radio Disabled

Agregue el atributo disabled (deshabilitado) para que los radio-default no se puedan seleccionar y los radio-checked se mantendran seleccionados

<!--Radio Tipo Normal-->
<div class="form-Radio-Normal-Default">
    <input class="form-check-input" type="radio" 
        name="flexRadioDefault" id="RadioDefault" disabled>
    <label class="form-check-label" for="RadioDefault">
        Normal Default radio
    </label>
</div>
        
<div class="form-Radio-Normal-Checked">
    <input class="form-check-input" type="radio" 
        name="flexRadioDefaultN" id="RadioChecked" checked disabled>
    <label class="form-check-label" for="RadioChecked">
        Normal Default checked radio
    </label>
</div>
