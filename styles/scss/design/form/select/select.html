Select

Personalice los "select" nativos con CSS personalizado que cambia la apariencia inicial del elemento Inspirados en los tipos de Pokemon

Default

Custom select menus need only a custom class, .form-select to trigger the custom styles. Custom styles are limited to the select’s initial appearance and cannot modify the options due to browser limitations.

<select class="form-select-Normal " aria-label="Default select example">
    <option selected>Open this select menu</option>
    <option class="op" value="1">One</option>
    <option class="op" value="2">Two</option>
    <option class="op" value="3">Three</option>
</select>

<select class="form-select-Fire " aria-label="Default select example">
    <option selected>Open this select menu</option>
    <option value="1">One</option>
    <option value="2">Two</option>
    <option value="3">Three</option>
</select>

<select class="form-select-Water " aria-label="Default select example">
    <option selected>Open this select menu</option>
    <option value="1">One</option>
    <option value="2">Two</option>
    <option value="3">Three</option>
</select>

Sizing

También puede elegir entre selecciones personalizadas de tamaños como pequeñas y grandes psolo añadiendoles las clases: select-lg o select-sm

<select class="form-select-Normal  " aria-label="Default select example">
    <option selected>Open this select menu</option>
    <option class="op" value="1">One</option>
    <option class="op" value="2">Two</option>
    <option class="op" value="3">Three</option>
</select>


<select class="form-select-Fire select-lg" aria-label="Default select example">
    <option selected>Open this select menu</option>
    <option value="1">One</option>
    <option value="2">Two</option>
    <option value="3">Three</option>
</select>

<select class="form-select-Water select-sm" aria-label="Default select example">
    <option selected>Open this select menu</option>
    <option value="1">One</option>
    <option value="2">Two</option>
    <option value="3">Three</option>
</select>

Disabled

Agregue el atributo disabled (deshabilitado) en una selección para darle una apariencia atenuada y eliminar eventos del select

<select class="form-select-Normal" aria-label="Default select example" disabled>
    <option selected>Open this select menu</option>
    <option class="op" value="1">One</option>
    <option class="op" value="2">Two</option>
    <option class="op" value="3">Three</option>
</select>
